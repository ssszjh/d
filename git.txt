1.下载git安装，默认就好
2.配置SSH KEY，去C:\Users\sss\.ssh，看有没有id_rsa和id_rsa.pub这两个文件
无则：在git里输入：$ ssh-keygen -t rsa -C xxx@qq.com 
你需要把邮件地址换成你自己的邮件地址，然后一路回车，使用默认值即可mi'shi
3.上GitHub里添加钥匙https://github.com/settings/ssh?，key为id_rsa.pub文件的内容，title随便写
4.使用命令“ssh -T git@github.com”，如果出现 You’ve successfully authenticated, but GitHub does not provide shell access 。 
这就表示已成功连上github。
5.创建Github仓库
6.在本地与Github仓库关联
其命令：
git init                 # 本地仓库初始化，执行完后会在工程目录下生成一个.git的隐藏目录
git remote add origin xxx   # 添加远程仓库地址，保存在origin变量中,弄一次就好，xxx为你仓库的clone
git add .               # 添加所有文件到本地索引，命令用法：git add <file>
git commit -m "x"   # 提交修改到本地仓库，-m选项添加提交注释
git push origin master      # 按照前一条命令中origin给定的github地址推送到github仓库的master分支
若git push origin master报错，则先执行git pull origin master ，先往下拉，再往上传，git push origin master
